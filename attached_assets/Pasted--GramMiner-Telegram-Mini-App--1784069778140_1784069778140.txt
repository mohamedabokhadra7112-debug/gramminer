اعمل التعديلات التالية على مشروع GramMiner (Telegram Mini App) دفعة واحدة:

═══════════════════════════════════
1) خلفية موحدة لكل صفحات التطبيق
═══════════════════════════════════

- الصورة المرفقة لازم تبقى خلفية عامة للتطبيق كله (كل الصفحات: 
  Mine, Miners, Tasks, Friends, Profile)، مش خاصة بصفحة واحدة بس.
- حطها على العنصر الأساسي (body أو أعلى Layout Component اللي 
  بيلف كل الصفحات/الراوتس)، مش داخل كل صفحة لوحدها.

CSS المطلوب:
body {
  background-image: url('/assets/background.png');
  background-size: cover;
  background-position: center;
  background-attachment: fixed;
  background-repeat: no-repeat;
  min-height: 100vh;
}

- شيل أي background منفصل كان متحط جوه كل صفحة لوحدها عشان 
  ما يبقاش فيه تعارض أو تكرار غلط للصورة.
- تأكد إن كل الكروت (user card, wallet card, إلخ) عندها خلفية 
  شبه شفافة (rgba داكن) فوق الصورة، عشان النصوص تفضل واضحة 
  في كل الصفحات من غير استثناء.
- لو التطبيق شغال بنظام Routing (React Router مثلاً)، حط الخلفية 
  على أعلى مستوى Layout Component، مش تكررها في كل Component.

═══════════════════════════════════
2) حماية الكود من النسخ/الفحص
═══════════════════════════════════

- امنع الضغط بالزرار اليمين (right-click) على الصفحة كلها:
  document.addEventListener('contextmenu', e => e.preventDefault());

- امنع اختصارات فتح أدوات المطور (F12, Ctrl+Shift+I, Ctrl+U, إلخ) 
  عن طريق event listener على keydown.

- اعمل Minify + Obfuscate لكل كود JavaScript قبل النشر (استخدم 
  مكتبة javascript-obfuscator) عشان الكود يبقى غير مقروء.

- انقل أي منطق حساس (حساب الأرباح، شروط الـ Claim، قيمة العملة، 
  التحقق من صحة المستخدم) بالكامل للباك إند، ومتخليش أي قيمة أو 
  معادلة حساسة مكتوبة في كود الفرونت إند.

- استخدم Environment Variables (Replit Secrets) لأي مفاتيح أو 
  توكنات، وأبدًا متكتبهاش مباشرة في الكود.

═══════════════════════════════════
3) تأكيد نجاح النشر على تليجرام
═══════════════════════════════════

- اعمل Auto-deploy on save عشان أي تعديل يتنشر فورًا.
- أضف رقم إصدار صغير في الفوتر (مثلاً v1.0.4) يتغير مع كل تعديل، 
  عشان أقدر أتأكد وأنا فاتح تليجرام إن التحديث فعلاً نزل.
- تأكد إن الـ Deployment URL ثابت (Static Deployment link) مش 
  بيتغير مع كل نشر، عشان رابط BotFather يفضل شغال.

نفذ التلاتة أجزاء دول مع بعض في نفس التعديل، وبعد ما تخلص أعمل 
build ونشر تلقائي على طول.
